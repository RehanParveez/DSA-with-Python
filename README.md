## Big-O-Notation
The concept of Big O notation is simply about helping us in measuring like how the performance of the algorithm is growing related to the increase of input size. So simply instead of calculating the exact time, it will rather focus on the rate of growth and the worst case performance scenario.

## Hashing:
 Hashing is a concept which allows the dictionaries and sets in python to be able to store and find the data extremely fast by converting the given keys into the specific indexes of array indexes with the use of a hash function.
